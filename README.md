STT는 사람의 말소리를 텍스트로 변환하는 기술입니다.
이는 음성 인식을 통해 구어체 언어를 캡처하고 이를 기록 가능한 형태의 텍스트로 변환합니다.
사용예시: 음성 명령 입력, 자동 회의록 작성, 음성 기반 검색 시스템 등.
